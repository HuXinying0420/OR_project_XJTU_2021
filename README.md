## 投资组合问题
某基金管理公司现有10000万元用于投资股市。假设在上证50的50支成分股中选择10种股票进行投资，以2019年9月间每种股票的日收盘价标准差作为风险度量，以8月30日至9月30日的涨幅作为预期收益。请为该基金公司设计最优投资方案（选出的10种股票的分别投资多少万元。
